# Uniwersum G.G.w.P

## Czym jest strona Uniwersum G.G.W.P
Jest to strona internetowa oparta o wspomniane unwiersum G.G.W.P rozwijana przez społeczność w aplikacji discord

## Co się tam znajduje?
Znajdują się tam takie rzczy jak:
- archiwum postaci
- archiwum lokacji
- archiwum światów
- archiwum przedmiotów
- archiwum wydarzeń
